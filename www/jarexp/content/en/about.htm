About

Jar Explorer - is a GUI tool for browsing java libraries known as JAR, WAR, EAR, APK, AAR and ZIP files.

Features

  • Supports Java 6 or higher
  • Can browse nested archives
  • Allows Drag&Drop adding, extracting
  • Allows popup menu adding, extracting and unpacking
  • Decompiles java classes with line numbers
  • Can identify java version which was compiled class file
  • Allows editing files inside archive
  • Allows review of content of folder or archive
  • Allows open and review separate .class file
  • Can search file in the archive or file system
  • Can search data in the file from archive or file system
  • Can search file duplicates in the archive or file system
  • Allows to get full command line of all Window's processes
  • Calculates MD5 and SHA-1 summs of any file from archive
  • Supports ICO files
