Installing on Debian Linux family hosts

Perform following command in terminal:

wget -O /tmp/jarexp_amd64.deb http://dst.in.ua/jarexp/downloads/jarexp_amd64.deb && sudo dpkg -i /tmp/jarexp_amd64.deb

Afterward type jarexp to run Jar Explorer
