О программе

Jar Explorer - это небольшая утилита для просмотра содержимого библиотек Java, известных как файлы JAR, WAR, EAR, APK и ZIP.

Возможности

  • Поддерживает Java 6 и выше
  • Можно работать с вложенными архивами
  • Умеет добавлять и извлекать содержимое с помощью Drag&Drop
  • Умеет добавлять, извлекать и распаковывать содержимое с помощью контекстного меню
  • Декомпилирует Java классы с сохранением номеров строк
  • Может распознавать какой версией Java был скомпилирован класс
  • Позволяет редактировать содержимое файлов внутри архива
  • Позволяет просматривать содержимое папки или архива
  • Умеет открывать .class файлы. Фактически работает как UI декомпайлер
  • Умеет искать файлы в архиве
  • Умеет искать по содержимому файлов в архиве
  • Умеет искать дубликаты файлов в архиве
